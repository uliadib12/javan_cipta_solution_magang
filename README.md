### ⚙️ Endpoint API

| Method | Endpoint | Deskripsi |
|---------|-----------|------------|
| **GET** | `/users` | Mengambil semua user |
| **GET** | `/users/:id` | Mengambil user berdasarkan ID |
| **POST** | `/users` | Menambahkan user baru |
| **DELETE** | `/users/:id` | Menghapus user berdasarkan ID |
